# <b>HTML ページの基礎</b>

### HTML5 の最小構成
```
<!DOCTYPE html>
<html lang="ja">
  <head>
    <meta charset="UTF-8">
    <title>xxx</title>
  </head>
  <body>
    Hello, world!
  </body>
</html>
```

### 各タグの説明
```
<!DOCTYPE html>
```
HTML5 文書に必須の文書型宣言。必ず HTML5 文書の先頭行に記述する。

```
<html lang="ja">
```
言語指定付の `<html>` タグ。
