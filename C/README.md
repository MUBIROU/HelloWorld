# C言語

## C言語について

* 1972年に登場した汎用プログラミング言語
* クラスベースのオブジェクト指向ではないが、構造体を利用した高度なプログラミングが可能
* 比較的[低水準](http://bit.ly/2meoB4s)な言語のため、コンピュータの仕組みに詳しくなるという理由で「初心者が学ぶべき言語」とも言われる
