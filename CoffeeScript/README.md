# CoffeeScript

## CoffeeScriptについて

* 2009年にリリースされた [Ruby](https://ja.wikipedia.org/wiki/Ruby) に似た AltJS（JavaScript の代替言語）
* JavaScript と比較すると簡潔に記述が可能であり、出力される JavaScript も綺麗である
* 単行のコメントはコンパイル時に削除される（ブロックコメントは残される）
