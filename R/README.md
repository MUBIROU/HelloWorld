# R言語

## R言語について

* 1996年に登場した統計解析専用のプログラミング言語（ ≠ 汎用プログラミング言語）
* クラスの作成法に ① S3 ② S4 ③ R5 ④ R6（追加パッケージが必要） がある
* 変数の定義は「変数<-値」とする
* 「.」に特別な意味はなく変数の命名にも利用可能
