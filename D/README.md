# D言語

## D言語について

* C言語をベースにしたオブジェクト指向および[関数型](http://bit.ly/1KTmmNW)のプログラミング言語
* C言語の後継を目指して開発（C言語との互換性はない）
* 2001年に Ver.0.001 が、2006年に Ver.1.0 のコンパイラが登場
* 2013年、Facebook の社内ツール開発に採用
